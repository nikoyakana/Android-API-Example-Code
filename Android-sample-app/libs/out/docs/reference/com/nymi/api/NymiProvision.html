Class Overview

Represents a Nymi Provision.

Contains methods to communicate with the band containing the provision to receive data specific to the provision. Each Nymi Band can contain up to six provisions. Instances of NymiProvision can be obtained only through getProvisions() and NymiAdapter.NewProvisionCallback.

If a null object parameter is passed to any method, then a NullPointerException will be thrown.

public void setTotpKey (String totpKey, boolean guarded, NymiProvision.TotpSetKeyCallback callback)

Set a key to be later used for for generating Time-based One Time Passwords.

A unique key needs to be set by the NEA before totp can be obtained. This key is expected to be provided to the NEA by a partner and does not change. This method needs to be run only once per provision, unless the key changes.

Parameters
totpKey partner totp key, sixteen characters long
guarded Set to true to require the band to be tapped prior to issuing the totp. Set to false to have the band issue the totp without requiring physical interaction.
callback callback through which the totp key will be delivered
