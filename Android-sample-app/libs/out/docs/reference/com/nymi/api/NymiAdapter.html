Class Overview

Represents the Nymi Adapter.

The NymiAdapter lets you perform fundamental Nymi tasks, such as initiate Nymi Bands discovery, provision new bands, query a list of provisioned bands, and send commands to provisioned bands.

Nymi Adapter can be used with a physical Nymi Band or with a nymulator. If used with a physical band, you will need BLUETOOTH and BLUETOOTH_ADMIN permissions. If used with a nymulator, INTERNET permission is required

Use the helper getDefaultAdapter() to get the default Nymi adapter for this Android device.

If attempting to use an uninitialized NymiAdapter an IllegalStateException will be thrown.

public void setPattern (BitSet agreementPattern)

Request the Nymi Band that is displaying the pattern to create a new provision and associate it with this adapter.

During the provisioning process, after the Nymi Band had established the initial connection with this Adapter, the Nymi Band will signal that it is ready to create a new provision by displaying an agreement pattern. This pattern will be received by the Nymi Adapter and forwarded to the application via the callback set through setAgreementCallback(NymiAdapter.AgreementCallback). The application can then present this pattern to the user to confirm.

Once the user confirms the pattern, the Adapter will request the Nymi Band to create a new provision and associate it with the Adapter. The Nymi Band will then send the provision id back and Adapter will inform of the new available provision through callback set via setNewProvisionCallback(NymiAdapter.NewProvisionCallback)

If the provided pattern is not set on the band, no callback will be invoked

Parameters
agreementPattern the pattern displaying on the intended provisioning peer
Throws
IllegalArgumentException if agreementPattern is null
